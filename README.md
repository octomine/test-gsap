# test-gsap

Плиточная стена в пекшоте перерисована руками в svg

## Scripts

- `yarn` установка пакетов
- `yarn dev` запуск для разработки
- `yarn build` сборка

## src/utils

Переиспользуемые элементы, можно даже попробовать применить в других проектах
